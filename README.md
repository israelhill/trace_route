# A trace route like program for Computer Networks: EECS 325

distMeasurement.py will calculate the number of hops between your machine
and a list web severs specified in targets.txt. This is done using only a single UDP
probe.

**To run distMeasurement.py:** sudo python distMeasurement.py

geoDistance.py will calculate the geographical distance between your machine and the
list of web servers specified in targets.txt.

**To run geoDistance.py:** python geoDistance.py
